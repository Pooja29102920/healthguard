# HealthGuard

Health monitoring app — real-time chat, SOS alerts, medicines, reports.

## Deploy to Railway (free, works on any device)

1. Go to [github.com](https://github.com) → New repository → name it `healthguard` → Create
2. Upload all these files by clicking **"uploading an existing file"** on GitHub
3. Go to [railway.app](https://railway.app) → Login with GitHub
4. **New Project** → **Deploy from GitHub repo** → select `healthguard`
5. Wait ~1 min → **Settings → Networking → Generate Domain**
6. Share that URL with everyone 🎉

## Run locally (same WiFi only)

```
cd healthguard
python server.py
```
Open http://localhost:8000
